GolangBCN's a local HQ for Gopher Gala 2016

Gopher Gala is a distributed global hackathon based on Go, happening next week. The event consists in a prized hackathon happening from January 23rd at 2:00 to January 25th 2:00, in which coders from all the globe will be hacking on chosen projects. Since the organizers behind GolangBCN had been long time discussing about taking part in some event involving practical programming - let’s say some action beyond such cool talks, this looked like the perfect excuse to measure our community’s taste for fun.
